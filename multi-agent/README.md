# multi-agent

Interactive, multi-agent storytelling over LiveKit. An intro agent gathers story details, then hands off to a genre-specific narrator (Sci-Fi, Fantasy, Noir) with realtime voice.

## Features
- Multi-agent handoff by genre
- LiveKit Agents server with STT/TTS
- Background typing audio while the model thinks

## Requirements
- Python 3.11+
- A LiveKit Cloud project (or self-hosted LiveKit)
- OpenAI API key
